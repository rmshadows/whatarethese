# VCF 通讯录整理工具

## 文件列表

- `1-vcf_to_csv.py`——vcf转csv
- `2-csv_to_vcf.py`——csv转vcf
- `m_VCF3.py`——VCF模块

## 使用

1. 首先从手机中导出VCF文件（需要3.0-4.0）
2. 使用1-vcf_to_csv.py将vcf转为csv
3. 新建Excel表格，导入csv的数据，开始整理
4. 整理完毕，Excel另存为csv文件，注意修改后面的vcf版本号等等数据（记得核对，比如有的手机仅支持导入vcf3.0）
5. 使用使用2-csv_to_vcf.py将csv转为vcf文件
6. 导入手机
