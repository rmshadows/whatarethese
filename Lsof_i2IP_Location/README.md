### 使用方法
- 版本：Python3.7
- 平台：Linux

- `sudo chmod +x Lsof_i2IP_Location.py`
- `./Lsof_i2IP_Location.py`

- 原理：

收集Linux下的`Lsof -i`命令查询到的与本机通讯的服务器IP地址。随即在[Chinaz平台](http://ip.tool.chinaz.com/)查询服务器名称。
